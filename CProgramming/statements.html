Program Statements

A statement must always end with a semicolon. Statements can be grouped together using { ... }.

Here are examples of the basic statement types. 
average = (x1 + x2 + x3)/3.0; Mathematical expression  

scanf("%d %lf",&k,&x); An input statement  

printf("%d %lf\n",k,x); An output statement  

assert(x >= 0); A debugging statement  

if((x > -3) && (x < +3)) { /*...*/ } Conditional statment  

for(k = 0; k < n; k++) { /*...*/ } Looping statement  

while(index > 0) { /*...*/ } Looping statement
