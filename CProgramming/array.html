Arrays

An array is a sequence of ints or doubles that are consecutive in the computer's memory. 
double vector[3] = { 1.0, 2.0, 3.0 }; variable declaration and initialization 
int daysOfTheMonth[12];
C arrays are indexed starting from zero. Be careful not to ask for non-existent array elements since your program will probably not complain, but will not work as expected.
You can used an indexed array element just like an int or double. 
summerDays = daysOfTheMonth[5] + daysOfTheMonth[6] + daysOfTheMonth[7]; /* Jun-Aug */  

if(leapYear) { conditional statement     
	daysOfTheMonth[1] = 29; 
}

C arrays are internally represented by their starting address in memory and the type (int/double) of each element. They do not know their intrinsic size, which can lead to some non-intuitive behavior if you are not expecting this.

When declaring a function that has an array input, the array size is optional. If you specify an array size, the compiler ignores it but this can still be useful to document what your function expects. Whether or not you include the array size in your function declaration, your code needs some way of knowing the array's size. Often the easiest solution is to add an explicit array size parameter. 
double dotProduct(double u[],double v[],int dim); function declaration
When passing arrays to a function, you just pass the array name without any indexing. 
double vector1[3],vector2[3]; 
dot = dotProduct(vector1,vector2,3);
