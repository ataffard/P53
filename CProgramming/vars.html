Declaring and Initializing Variables

A variable must be declared before it can be used. A declaration specifies a variable's name and the type of quantity that the variable will hold. Two types are sufficient for almost all numerical work: int and double. 
int k; 
double centroidX; Other possible types are listed under advanced features
You can declare several variables of the same type in one statement. 
int i,j; 
double centroidY,centroidZ;
A variable can be used in your program in lines following its declaration, up to the first un-matched closing bracket }. The program lines where a variable can be used are called its "scope". 
{     
	{         
		double theta;         
		{ /* A */             
			{ /* B */             
			} /* balances B */         
		} /* balances A */     
	} /* first unbalanced: scope of theta ends here */ 
}
Variable names should generally be self-describing.
The only exception is when a variable's function is obvious from how it is being used (as the index of a for loop, for example).

Names cannot contain spaces but multi-word names can be written as myFirstVariable or my_first_variable (pick one of these conventions and stick with it).
Names can use a-z, A-Z, 0-9 and the underscore character "_".
Names cannot start with a digit 0-9.
By convention, names start with a lower-case letter.
Names must be unique within their scope (but case matters) and you cannot use reserved keywords for names (if,for,while,...)

You can declare and set an initial value for a variable at the same time.
You can mix initialized and un-initialized variables in one statement.
If you declare a variable but do not specify its initial value, it will often be zero but this is not guaranteed in C and you should never rely on this. 
int n,size=3; 
double pi=3.14,area;
The value of a double can be specified using scientific notation. 
double speedOfLight=2.9979e8;
You can make an array of either type (int,double). The array size must be a fixed integer constant. 
int daysOfTheMonth[12]; An array of values 
double positionVector[3]; An array of values
You can specify initial values for the elements of an array using the following syntax: 
int velocity[3] = { v0*cos(theta), v0*sin(theta), 0 };
