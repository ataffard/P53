Math Routines

Math routines are defined in the math.h header.

The trig routines (cos,sin,tan,acos,asin,atan,atan2) work in radians. 
x = cos(theta); 
y = sin(theta); 
theta = atan2(y,x);
Use pi/180. to convert from degrees. 
pi = 4*atan(1); 
deg2rad = pi/180; 
theta = 45*deg2rad;
There are several different ways to convert a double x to a nearby int value. The (int) appearing in front of each expression below is called a "cast" and is necessary to force the compiler to perform a conversion that potentially loses information. The ceil, floor functions are defined in the math.h header. 
(int)x;        /* rounds towards zero */ 
(int)ceil(x);  /* rounds towards +infinity */ 
(int)floor(x); /* rounds towards -infinity */
Floating-point expressions can return special values to indicate an infinite (+inf,-inf) or undefined (nan) value. 
bad = sqrt(-1); 
assert(bad == bad); /* "nan" is the only value that fails this test */
The math library also defines hyperbolic functions (cosh,sinh,tanh) and power functions (exp,log,log10,pow,sqrt) and a floating-point absolute value (fabs).
