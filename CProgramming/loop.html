Looping Statements

Use a for loop when you know in advance how many times you want to repeat. The loop variable should be an int that you have already defined. 
for(k = 0; k < n; k++) { k < n can be any conditional expression     
	/* repeat this stuff n times with k = 0,1,...,n-1 */ 
}
Use a while loop when you don't know in advance how many times to repeat or you don't need an index variable. 
while(x > 0) { x > 0 can be any conditional expression     
	/* repeat this stuff as long as x > 0 */ 
}
In case you always want to execute a loop at least once, use a do-while loop. 
do {     
	/* this will execute at least once */ 
} while(x > 0); x > 0 can be any conditional expression
Loops can be nested. You can nest for loops within while/do-while loops and vice-versa.

Within any loop, there are special statements that will jump to the next loop iteration (continue) or else jump right out of the loop. 
for(i = 0; i < n; i++) {
     for(j = 0; j < n; j++) {
         if(i == j) continue;         
		  /* remaining statements only execute when i != j */     
	  } 
}  

while(x > 0) {     
	/* ... */     
	if(y < 0) break;     
	/* ... */ 
}
