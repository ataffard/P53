Testing A Condition

Use the if statement to selectively execute statements. 
if(x > 0) { test any logical expression     
	/* these statements only execute if the condition is true */ 
} 
else {     
	/* these statements only exeute if the condition is false */ 
}
The else clause can be omitted. 
if(x > 0) { test any logical expression     
	/* ... */ 
}
Mulitple if clauses can be chained together, with our without a final else clause. 
if(x > 0) { test any logical expression     
	/* ... */ 
} 
else if(x < 0) { test any logical expression     
	/* ... */ 
} 
else {     
	assert(x == 0); better safe than sorry 
}
