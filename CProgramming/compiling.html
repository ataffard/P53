Compiling

To compile your code in C, name your file with the .c extension and use:
gcc -Wall -o yourCode yourCode.c

To compile your code in C++, name your file with .cpp extension and use: 

g++ -Wall -o yourCode yourCode.cpp

-Wall: All warning will show up during compilation time.
-o: indicate the name of the output excecutable
the last argument is your code.

Note that there are differences in syntax between C and C++.
