Input and Output

The printf and scanf I/O routines are defined in the stdio.h header file.

Use %d for int values and %f for double values (use %lf when reading in). Each %d or %f must match up with a variable of the appropriate type.
For output, use \n to start a new line. It can be useful to leave it out for an input prompt.
For input, remember to put an ampersand (&) in front of each variable. 
printf("Hello, world\n"); /* No variable values are printed */  
printf("Enter a number: "); /* Leave out the \n for an input prompt */  
scanf("%lf %lf",&x,&y); /* Remember the ampersand (&) with scanf */  
printf("x = %f, y = %f\n",x,y);
To read input from a file or save output to a file, use I/O redirection when you run your program. 
myprog < input.dat 
myprog > output.dat 
myprog < input.dat > output.dat
The scanf function returns the number of values that were successfully captured from the input stream.
It is safe to ignore this but it can also be useful for error checking.
The special value EOF is returned when scanf encounters the end of a file. 
int converted;  

converted = scanf("%lf %lf",&x,&y); 

if(converted == 2) {     
	/* input was scanned successfully */ 
} else if(converted == EOF) {    
	/* end of redirected input file was reached normally */ 
} else {  
	/* found a line that could not be scanned */ 
}
