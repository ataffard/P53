Mathematical Expressions

The familiar arithmetic operators are *, /, +, -. There is also a modulus operator % that only works for int's. 
a*x*x + b*x + c 
index % n
When a binary operator has mixed int and double arguments, the int will be automatically promoted to a double for the calculation. In case both arguments to a division are int's, the result will also be an int that discards any fractional part. 
double x;
x/3; /* floating-point division (3 -> 3.0)*/ 
int k; 
k/3; /* integer division */
You can use + and - as "unary" operators (unary + sometimes makes your program easier to read even if it is unnecessary) 
+(x-y) 
-b + 4*a*c
You can use a mathematical expression anywhere that a number is expected. 
int k = i+j; 
isPrime(i+j) 
printf("%lf\n",x+y);
