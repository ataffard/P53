Pointer

Pointer is different type of variable. A pointer contains the address of the variable it points to.

Remember to always initialize your pointer to either the address of a declared variable or to NULL.

int x = 4;
int *xPtr=NULL; //declare a pointer to an int and set it to NULL

xPtr = &x;      //make xPtr point to x (ie set the address in xPtr to the address of x using the address operator &

*xPtr = 10;     //Change the valu of the variable pointed to by xPtr (ie x). Equivalent to x=10;
