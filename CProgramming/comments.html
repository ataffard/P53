Comments

A comment is text intended for human readers and is ignored by the compiler. 
/* A comment can be on a single line, like this */  
double pi = 3.14; /* A comment can start anywhere on the line */ 

/* A comment can be spread 
across 
several 
lines */  

double G=6.47e-11; //Universal Gravitaional Constant (The rest of the line can be commented out)
Note that // is a C++ comment format
