Functions

A function must be declared before it is used.
A declaration specifies what the function returns (int, double, or void if it doesn't return anything) and what types of input values it expects to receive. 
int daysOfMonth(int month); 
void swap(int* a, int*b);   
double polygonArea(double x[],double y[],int nVertex);
A function initialization starts like a declaration, but replaces the semicolon with the statements that the function will execute. 
double myFunction(double x, double y) {     
	Define your local variables here    
	Add your function statements here 
}
Your function statements must return a value of the appropriate type. You can have multiple return statements. 
int daysOfMonth(int month) {     
	assert((month >= 1) && (month <= 12)); better safe than sorry     
	
	if(month == 1) { conditional statement         
		return 31; /* 31 is an int value */    
	}     
	else if(month == 2) {         
		return 28; /* ignore leap years */     
	}     
	/* ... */ 
}
An initialization also declares your function so you don't need a separate declaration statement earlier in your program, except in cases where its not possible to initialize each function before it is used, as in this example. 
double g(double x);
  
double f(double x) {     
	/* calls g(x) */ 
}  

double g(double x) {     
	/* calls f(x) */ 
}
Statements defined within a function can refer to any local variables that have already been defined in the function as well as the input variables listed in the function declaration. In case you define a local variable with the same name as an input variable, the local variable will hide the input variable which is usually not what you want.

Function inputs are passed to your function as values.
Any change you make to these values will be lost when your function returns to its caller.
An important caveat is that you can make permanent changes to array contents. 
void arrayChanger(double vector[]) { array input     
	vector[0] = vector[1] + vector[2];     
	return; 
}



If your function needs to modify the input parameters and return the modified value, you need to use pointers.

void swap(int* a, int* b){|
	int temp = *a;
	*a = *b;
	*b = temp;
}
